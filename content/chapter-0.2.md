# Chapter 0.2 — Installation Guide

## 0.2.0 Overview

This chapter provides a step-by-step guide for installing and configuring OpenClaw on a Windows environment.

### 0.2.1 Step 1: Operating System Setup

*(requirement)* **Windows Installation:** The primary environment for OpenClaw is Windows. Ensure you have a clean installation of Windows 10 or 11. \
*(setup)* **Windows Account:** During the initial setup, you will be prompted to sign in with a Microsoft account. Use the following Gmail address as a placeholder for the account credentials. \
*(copyable)* **Account Gmail:**
```text
windows.setup@gmail.com
```
\
*(note)* **Verification:** Ensure that the Windows installation is fully updated and all drivers are correctly installed before proceeding to the OpenClaw setup.
